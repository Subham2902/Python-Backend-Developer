# Day 1 - Introduction to Python

## What is Python?

Python is a high-level, interpreted, and easy-to-read programming language used for:

- Web Development
- Backend Development
- Artificial Intelligence (AI)
- Machine Learning (ML)
- Data Science
- Automation
- Scripting

---

## Installing Python

- Download Python from the official website.
- Install Python.
- Verify installation:

```bash
python --version
```

or

```bash
python3 --version
```

---

## Running Python

### Interactive Mode

```bash
python
```

### Python File

```bash
python hello.py
```

---

## First Python Program

```python
print("Hello, World!")
```

### Output

```
Hello, World!
```

---

## print() Function

Used to display output on the screen.

Example:

```python
print("Welcome to Python")
print(100)
print(True)
```

---

## Comments

Single-line comment

```python
# This is a comment
```

Multi-line comment

```python
"""
This is
a multi-line comment.
"""
```

---

## Keywords Learned

- print()
- Comments

---

## Key Points

- Python is case-sensitive.
- Indentation is important.
- Every Python program starts executing from top to bottom.
- Practice by typing code yourself instead of copy-pasting.
